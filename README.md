# Lykos Method — Landing

Sitio oficial de **Lykos Method**, el método de entrenamiento online de Enzo Nievas.
Entrená con un método que respeta tu cuerpo — diagnóstico, planificación y seguimiento
personalizados, desde Mendoza al mundo.

## Qué ofrece

- **Tres intensidades** — alta, media y baja, según el objetivo del alumno.
- **Cinco pasos del método** — anamnesis → planificación → feedback → seguimiento quincenal → balance mensual.
- **Seis perfiles de alumno** — alto rendimiento, rehabilitación, reeducación muscular, adultos mayores, principiantes, salud y bienestar.
- **Tres modalidades** — rutina única, planificación mensual con seguimiento, plan integral con red de profesionales.

## Stack

Single-file HTML + CSS + JS vanilla.

- **Three.js** (via `esm.sh`) — columna vertebral 3D en la sección *Anatomía*.
- **GSAP + ScrollTrigger** — rotación de la columna ligada al scroll.
- **Google Fonts** — Fraunces (display), Hanken Grotesk (body), JetBrains Mono (labels).
- Bilingüe ES/EN con switcher en vivo.
- Formulario de contacto → WhatsApp (handoff con mensaje prefilled).

## Deploy

GitHub Pages sirve `index.html` directamente desde la raíz de la branch `master`.
